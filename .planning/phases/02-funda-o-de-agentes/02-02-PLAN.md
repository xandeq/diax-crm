Extend the existing raw-HttpClient `AnthropicChatClient` with a non-breaking, opt-in third method `CompleteWithToolsAsync` that supports Anthropic tool-use (function-calling), returning either a `TextResponse` or a `ToolUseRequest` (ORCH-02 — shared engine, tools differ per agent).
